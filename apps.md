# Apps

Configs in this repo:

- Alacritty (terminal emulator) — `stow/alacritty`
- AwesomeWM (window manager) — `stow/awesome`
- Bash config — `stow/bash`
- Codex CLI — `stow/codex`
- Fontconfig — `stow/fontconfig`
- X11 config — `stow/x11`
- WezTerm (terminal emulator) — `stow/wezterm`
- Zellij (terminal workspace) — `stow/zellij`

Keep this list in sync with `stow/` packages.

Dependencies:

- `jq`
- `libnotify`
- `stow`

Installed via `scripts/setup.sh`.
